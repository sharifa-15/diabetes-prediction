# 🩺 Diabetes Prediction using Machine Learning

## 📌 Project Goal
Predict the risk of diabetes using patient medical records (Pima Indians Diabetes dataset).

## 📂 Dataset
- Source: UCI Machine Learning Repository
- Features: Pregnancies, Glucose, BloodPressure, SkinThickness, Insulin, BMI, DiabetesPedigreeFunction, Age
- Target: Outcome (1 = diabetic, 0 = non-diabetic)

## 🔎 Workflow
1. Data Preprocessing (handle missing values, normalize features)
2. Exploratory Data Analysis (EDA: histograms, heatmaps, boxplots)
3. Model Training (Random Forest, Logistic Regression, XGBoost)
4. Evaluation (Accuracy, Precision, Recall, F1-score, ROC-AUC)
5. Feature Importance (Glucose, BMI, Age are strong predictors)
6. Ethical Considerations (privacy, fairness, transparency)

## 📊 Results
- Random Forest achieved ~80% accuracy
- Glucose and BMI are top predictors

## 🚀 How to Run
- Open the notebook in Google Colab
- Upload `diabetes.csv`
- Run cells step by step

## 🛡️ Ethical Note
This project is for educational purposes. In real healthcare applications, patient privacy and fairness must be strictly ensured.
